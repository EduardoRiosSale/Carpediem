Carpediem
Carpediem es una app en de gestión para gimnasios que desarrollé como proyecto personal. Permite administrar alumnos, rutinas, clases y ventas desde un solo lugar, con acceso diferenciado según el rol de cada usuario.
Funcionalidades

El Owner tiene control total: agrega usuarios, gestiona productos, ventas, ve métricas de alumnos y administra las calificaciones
El Profesor crea y asigna rutinas personalizadas a cada alumno
El Alumno puede ver su rutina, las clases disponibles y los servicios del gimnasio
Hay un chat interno entre Profesor --> Alumno y Owner ---> Alumno
Los alumnos pueden dejar calificaciones

Tecnologías que use

Frontend: React + TypeScript
Backend: Node.js + TypeScript
Base de datos: PostgreSQL
